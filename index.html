Bookmarklet for Jobbole

本书签工具主要功能是方便用户向伯乐在线 | 头条分享文章，虽然官方也提供了浏览器扩展，个人觉得安装扩展显得有些“笨重”，而且最主要的是消耗系统资源。于是，参考 pocket 与 一点黑科技，写了本书签工具，希望对大家有帮助。 😊

使用方法

将下面的+ Jobbole 链接拖到浏览器的书签栏即可。

+ Jobbole

点击该书签工具后，当前所在的页面会被自动提交到伯乐在线投稿页面。

该书签工具会自动填写标题、网址和摘要，做到真正的快捷高效。

摘要优先使用当前选中的文本，如果没有就去查找 meta 标签中的 description
